# ProgrammingQuotes 
React WebApp to exercise fetching data form Apis.

using [CC0 Avatar placeholder api.](https://pravatar.cc/)<br />
using [ProgrammingQuotesApi.](https://programming-quotes-api.herokuapp.com/index.html)

 
To run the project:

1. Clone this repo
2. Run `npm install`
3. Run `npm start`
